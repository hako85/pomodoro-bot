🍅 Pomodoro Bot - README

🌟 機能一覧

  • VCに参加し、ポモドーロタイマーを開始
  • Focusタイム中は全員強制ミュート
  • Breakタイム中はミュート解除
  • Focusタイム中にYouTube音楽再生（途中停止・途中再開対応）
  • 設定内容はsettings.jsonに保存され、次回起動時に引き継ぎ

📝 使い方

  • !start : タイマー開始
  • !stop : タイマー停止
  • !leave : VCから退出
  • !set <key> <value> : 設定変更
    例：!set focus 30 / !set music_url https://youtu.be/xxxx
    対応キー：focus, break, longbreak, interval, music_url
  • !status : 現在の設定と状態確認

⚙️ 動作仕様

  • Focusタイム：指定した分数だけ全員ミュート＆音楽再生
  • Breakタイム：全員ミュート解除
  • Interval回数分のFocus/Break終了後、Long Breakタイムが開始
  • settings.json内で設定値・Botトークンを管理
  • コマンド入力ミス時は丁寧にエラーメッセージを表示

💡 注意事項

  • 音楽再生には ffmpeg, yt_dlp が必要です（インストール済みならOK）
  • 一部著作権制限付きのYouTube動画は再生できない場合があります
  • 多人数・複数サーバー対応が必要な場合は Lavalink版を推奨

🍅 Happy Pomodoro Life! 🍅
